# 江欣苑 3D 打印混凝土板块——材质分析

> 项目：汉阳区江欣苑老旧社区第四期改造——"友好花园"  
> 团队：湖北工业大学土木建筑与环境学院 黄艳雁教授团队  
> 3D 混凝土打印主设人：洪波  
> 分析范围：项目中使用的四种材质——**塑木、白色混凝土、灰色混凝土、黑色混凝土**

---

## 一、材质总览与设计策略

江欣苑 3D 打印专题共设计生产了 **5 类 22 件城市家具产品**（座椅 17 件、花盆 5 式样 6 个、汀步石 43 块），在材质选择上并非使用单一配方的混凝土，而是采用了 **"三色混凝土 + 木复合"** 的四材质体系。这一策略体现了从 **"能打印"** 到 **"好设计"** 的跨越——材质不再仅仅是结构载体，而成为空间叙事、用户感知和社区美学的重要组成部分。

| 编号 | 材质 | 应用产品 | 核心属性 | 设计角色 |
|:---:|------|----------|----------|----------|
| 1 | **塑木**（木复合） | 座椅面板（塑木垫板） | 温润、低导热、触觉友好 | **人体界面层**——解决混凝土"冷硬"短板 |
| 2 | **白色混凝土** | 汀步石（悦步石）、部分花盆 | 明快、轻盈、引导视线 | **视觉引导层**——在老旧社区暗沉基调中"提亮" |
| 3 | **灰色混凝土** | 方隅座椅、主体结构基底 | 质朴、中性、结构感 | **结构基础层**——打印主材，承载力学与体量 |
| 4 | **黑色混凝土** | 动物造型花盆（河马/小狗/小猫/小龙）、适老花盆 | 稳重、雕塑感、衬托植被 | **造型表达层**——强化仿生形态的轮廓辨识度 |

## 二、材质逐一分析

### 2.1 塑木（Wood-Plastic Composite / 木复合面板）

#### 2.1.1 在项目中的应用

- **产品**：猫咪座椅（2件）、绿化座椅（2件）、方隅座椅（13件）的面层
- **规格**：400mm 宽塑木垫板，铺设在 3D 打印混凝土座椅胎体之上
- **功能标注**：项目报告注明"塑木垫板——增加舒适度"

#### 2.1.2 材质特性分析

| 维度 | 说明 |
|------|------|
| **热工性能** | 塑木导热系数（约 0.2–0.3 W/m·K）远低于混凝土（约 1.4–1.8 W/m·K），在武汉冬冷夏热气候下，坐感温度差可达 8–15°C——这是"适老化"的核心细节：老人对低温更敏感，直接坐于混凝土面会产生强烈不适 |
| **触觉体验** | 塑木表面具有微弹性与木质纹理触感，消除了混凝土"又冷又硬又糙"的负面感知，大幅提升使用意愿 |
| **耐久与维护** | 相比天然木材，塑木（木粉/稻壳 + HDPE/PP 共挤）不腐烂、不虫蛀、免漆维护，适合武汉高湿环境；天然实木在户外 2–3 年即出现开裂、霉变 |
| **色彩衔接** | 木色在视觉上形成"暖色—中性灰"的调和，避免全混凝土的"冷感"，与社区花园的自然语境协调 |
| **施工方式** | 后安装（非一体打印），通过预留在混凝土胎体上的安装孔位固定，便于日后更换受损面板 |

#### 2.1.3 设计意义

塑木垫板是项目中 **"材质互补"策略的核心体现**：3D 打印擅长造型但不擅表面触感，传统木工擅长表面但不擅异形——二者结合，实现了"复杂形态 + 宜人触感"的兼得。这一做法契合 **"全龄友好"** 设计理念中"老人看护儿童时可舒适久坐"的场景需求（项目报告明确将 3D 打印座椅定位为"供老人看护儿童使用"）。

### 2.2 白色混凝土（White Concrete）

#### 2.2.1 在项目中的应用

- **产品**：悦步石（3D 打印汀步，43 块）、部分花盆（推断用于点缀性构件）
- **经济技术指标**（汀步石）：材料用量约 6 kg/块，打印时长约 0.5 h/块，数量 43 块

#### 2.2.2 材质特性分析

| 维度 | 说明 |
|------|------|
| **胶凝材料** | 以白水泥（主要成分为白色硅酸盐水泥，Fe₂O₃ 含量 < 0.5%）替代普通灰水泥，白度可达 85–92%，成本约为普通水泥的 2–3 倍 |
| **骨料选择** | 需配合白色或浅色细骨料（如白色石英砂、白云石砂），避免普通河砂中的暗色矿物污染白色调 |
| **流变性能调控** | 白水泥早期水化热较低，凝结时间偏长，需外加促凝组分或流变调节剂（如温轮胶、纤维素醚）来维持挤出后的站立稳定性，这对小尺寸构件（汀步石仅 6 kg）的快速成型至关重要 |
| **视觉效果** | 在老旧社区暗沉地面（灰色水泥地、沥青路）上，白色汀步石形成**高对比度路径引导**，增强空间方向感和安全感，夜间也更易被识别——这在"通行安全便捷"评价维度上有明确贡献 |
| **耐污性挑战** | 白色表面在户外环境中更易显脏（雨渍、泥点、落叶渍），需考虑表面封闭处理（如硅烷浸渍）以降低吸水率和污渍附着 |

#### 2.2.3 设计意义

白色汀步石（"悦步石"）在亲子乐园组团中充当**地面视觉节奏的"标点符号"**，引导儿童在花园中探索行走，同时在实际功能上提供防滑（打印纹理天然具有凹凸肌理）、不积水（可设计排水坡度）的步道体验。白色混凝土的轻盈感也消解了 43 块汀步石可能带来的体量压迫。

---

### 2.3 灰色混凝土（Gray Concrete）——项目主力打印材料

#### 2.3.1 在项目中的应用

- **产品**：方隅座椅（13 个）、猫咪座椅和绿化座椅的胎体、各类产品的结构基层
- **用量**：单个方隅座椅约 50 kg/件，猫咪座椅约 75 kg/件，绿化座椅约 100 kg/件
- **打印时长**：方隅座椅 2 h/件，猫咪座椅 4 h/件，绿化座椅 8 h/件

#### 2.3.2 材质特性分析

| 维度 | 说明 |
|------|------|
| **胶凝体系** | 以普通硅酸盐水泥（P·O 42.5 或 P·O 52.5）为基材，辅以粉煤灰、矿粉或硅灰等矿物掺合料调节流变性与早期强度 |
| **骨料体系** | 因打印喷头口径限制（通常 10–30 mm），最大骨料粒径一般控制在 4–8 mm 以下，本项目小型城市家具构件预计采用细骨料砂浆体系（最大粒径 < 2 mm），属"打印砂浆"而非传统意义上的"混凝土" |
| **关键外加剂** | ① 聚羧酸减水剂——调节用水量与流变性，坍落度提升 30%–40%；② 缓凝/促凝组分——控制可打印时间窗口（Open Time）；③ 触变剂（如纳米黏土、硅灰）——赋予材料"挤出时流动、挤出后站立"的触变特性 |
| **力学性能** | 作为非结构城市家具，设计抗压强度预估在 C25–C40 等级，核心力学指标为**层间粘结强度**（通常为整体强度的 60%–85%，是薄弱环节） |
| **表面肌理** | 保留自然的打印层纹（层高约 6–15 mm），形成独特的技术美学——这种"诚实"地暴露制造痕迹的做法，反而是 3D 打印产品的辨识度来源 |
| **成本优势** | 灰色混凝土是三种混凝土中成本最低的，水泥为市场大宗材料（约 400–500 元/吨），无需额外着色处理 |

#### 2.3.3 配合比参考（基于文献推断）

| 组分 | 预估掺量范围 | 作用 |
|------|:---:|------|
| 普通硅酸盐水泥 P·O 42.5 | 基准 | 胶凝主体 |
| 粉煤灰 / 矿粉 | 10%–30% | 改善流变性、降低成本、减少水化热 |
| 硅灰 | 2%–5% | 提高触变性、增强层间粘结 |
| 细骨料（0–2 mm） | 胶砂比 1:1 ~ 1:2 | 骨架 |
| 聚羧酸减水剂 | 0.2%–0.5% | 降粘、保坍 |
| 纤维素醚 | 0.05%–0.15% | 保水、增稠 |
| 水胶比 | 0.30–0.40 | 控制可挤出性与早期强度 |

#### 2.3.4 设计意义

灰色混凝土在项目中扮演 **"沉默的背景"**——它承载了大部分座椅的结构功能，以中性灰色调和社区现有建筑色调（水泥抹灰墙面、灰色铺地）形成视觉延续，不争不抢，让白色汀步的轻盈和黑色花盆的雕塑感得以凸显。方隅座椅以 50 kg/件的轻量化和 2 h/件的快速打印，体现了灰色混凝土体系在**标准化批量生产**中的效率优势。

### 2.4 黑色混凝土（Black Concrete）

#### 2.4.1 在项目中的应用

- **产品**：河马花盆、小狗花盆、小龙花盆、小猫花盆、适老花盆（共 5 式样 6 个）
- **用量**：单个花盆约 25 kg/件（适老花盆为 25 kg/件）
- **打印时长**：2–2.5 h/件

#### 2.4.2 材质特性分析

| 维度 | 说明 |
|------|------|
| **着色方式** | 主要有两种路径——① **氧化铁黑颜料**（Fe₃O₄，掺量 3%–8% 水泥质量）直接混入干混料，工艺简单，成本低，但可能轻微影响流动性和强度（通常控制在 5% 以内影响不显著）；② **炭黑/碳纳米管**着色，颜色更深沉但分散难度大，需配合超声分散或预混色浆 |
| **着色对性能的影响** | 文献表明，氧化铁颜料（掺量 < 5%）对抗压强度的负面影响通常 < 10%，在非结构城市家具（花盆）中可接受；颜料颗粒的细度（通常 < 1 μm）可能增加用水量，需微调减水剂掺量以维持打印性 |
| **热工特性** | 黑色表面吸收太阳辐射（太阳吸收率 α ≈ 0.85–0.95），在武汉夏季可使表面温度比灰色高 10–20°C——这对花盆而言反而是**优势**：较高土温可促进部分喜温植物（如多肉、香草类）生长，延长春秋季室外植物的观赏期 |
| **造型衬托** | 动物造型花盆（河马、小狗、小猫、小龙）的设计依赖于**清晰的轮廓剪影**来传达"象形"识别——黑色混凝土通过强明暗对比，最大化地凸显了仿生形态的边界，远观效果好 |
| **与植被的色彩关系** | 黑色盆体与绿色植物（本项目配置的金叶石菖蒲、翠芦莉、果汁阳台月季等）形成经典的**黑-绿对比**，使植物色彩更饱和，花盆本身退为"画框"角色 |
| **耐污与老化** | 黑色相比白色更耐脏污，户外维护频率低；但氧化铁黑在强紫外线长期照射下可能轻微褪色（向棕褐色偏移），5–10 年周期内需考虑 |

#### 2.4.3 设计意义

黑色混凝土的五款造型花盆是项目中最具 **"情感温度"** 的产品线——河马、小狗、小猫、小龙的动物造型明显面向儿童群体，契合"亲子乐园"组团的定位。黑色赋予花盆以**雕塑感和体量感**，使 25 kg 的轻量花盆在视觉上"沉"下来，避免轻飘的塑料感。适老花盆则采用抬升高度设计（从项目尺寸图可见约 1050mm 高，适合老人站立操作），黑色底座稳重安全，体现适老化细节。

## 三、四材质体系的设计逻辑总结

```
                    【视觉光谱：从亮到暗】
                      ↑
        白色混凝土    │    ┌─ 路径引导（汀步）
        （明快轻盈）   │    └─ 空间提亮
                      │
        塑木          │    ┌─ 触觉温暖（座椅面层）
        （温润中性）   │    └─ 全龄友好
                      │
        灰色混凝土    │    ┌─ 结构基础（座椅胎体）
        （质朴中性）   │    └─ 成本与效率最优
                      │
        黑色混凝土    │    ┌─ 造型表达（花盆）
        （沉稳雕塑）   │    └─ 情感与识别
                      ↓
```

| 设计维度 | 材质策略 | 对应的社区需求 |
|----------|----------|:--------------|
| **触觉舒适** | 塑木垫板覆盖混凝土胎体 | 老人久坐不冷、儿童触摸不糙 |
| **视觉引导** | 白色汀步石在地面形成亮色节点 | 通行安全、夜间可辨 |
| **结构经济** | 灰色混凝土承担主要力学体量 | 项目预算可控（3D 打印总计 10.05 万元） |
| **情感识别** | 黑色花盆强化仿生轮廓 | 儿童亲近自然、亲子互动场景 |
| **环境融合** | 四色体系形成"暗-灰-木-白"的渐变 | 不突兀于老旧社区，但足以焕新 |

## 四、材质创新点与学术价值

### 4.1 将"颜色"从装饰提升为功能要素

大多数 3DCP 研究聚焦于力学性能和可打印性，颜色仅被视为后期涂装。江欣苑项目将 **白色/灰色/黑色的材料级着色**（而非表面涂装）融入设计策略——路径引导（白色汀步）、造型识别（黑色花盆）、环境融合（灰色基底），使颜色成为空间功能的一部分。

### 4.2 "混凝土 + 木复合"的人本思路

该做法回应了 3DCP 城市家具的核心短板——**混凝土的触觉排斥**。以打印胎体解决形态问题，以塑木面板解决界面问题，为"3D 打印如何落地为人所用"提供了一个低成本、高效果的答案。

### 4.3 彩色混凝土在社区场景的示范效应

三色混凝土的同时应用为"彩色装饰砂浆在 3D 打印中的应用"提供了小型工程实证数据——不同颜料的掺量、对打印性的影响、户外耐久表现，均可为后续研究和规模化推广提供参考。

## 五、改进建议

| 问题 | 建议 |
|------|------|
| 四材质的长期户外对比数据缺失 | 建议在江欣苑开展 1/3/5 年定期巡检，记录各色混凝土褪色、开裂、污渍累积情况，积累耐久性数据 |
| 白色汀步石耐污性 | 表面施加无色硅烷/硅氧烷浸渍封闭，降低吸水率至 < 2% |
| 塑木面板与混凝土胎体的连接耐久性 | 监测金属连接件锈蚀、塑木翘曲变形，探索一体打印的"混凝土肌理坐面"作为低成本替代方案 |
| 黑色花盆夏季土温过高 | 可在盆体内壁增加 5mm 气凝胶隔热涂层或预留空气夹层 |
| 配合比未公开 | 建议发表配合比设计论文，为后续社区级 3DCP 推广提供可复用的"配方包" |
